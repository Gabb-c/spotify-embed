## Sizes

Changing the `size` of the embed is easy. It can be `default, compact or custom`:

### Default

```js
import React from 'react';
import { SpotifyEmbed } from 'spotify-embed';

function App() {
  return (
    <>
      <SpotifyEmbed src="https://open.spotify.com/playlist/37i9dQZF1DX5trt9i14X7j" size="default" />
      {/*
        width: 300
        height: 380
      */}
    </>
  );
}

export default App;
```

### Custom

When using `custom` in the `size` property,
it will take the values from `width and height` properties:

```js
import React from 'react';
import { SpotifyEmbed } from 'spotify-embed';

function App() {
  return (
    <>
      <SpotifyEmbed
        src="https://open.spotify.com/playlist/37i9dQZF1DX5trt9i14X7j"
        size="custom"
        width={600}
        height={760}
      />
      {/*
        Default
        width: 300
        height: 380
      */}
    </>
  );
}

export default App;
```
