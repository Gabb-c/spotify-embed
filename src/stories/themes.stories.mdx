## Themes

Changing the `theme` of the embed is easy. It can be `light or dark`:

### Light (default)

```js
import React from 'react';
import { SpotifyEmbed } from 'spotify-embed';

function App() {
  return (
    <>
      <SpotifyEmbed src="https://open.spotify.com/playlist/37i9dQZF1DX5trt9i14X7j" theme="light" />
    </>
  );
}

export default App;
```
