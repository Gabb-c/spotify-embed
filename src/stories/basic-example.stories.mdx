## Installation

```bash
# NPM
$ npm install spotify-embed

# Yarn
$ yarn add spotify-embed
```

## Basic Usage

Import the component in your app and pass a **spotify resource URL/URI** in the **src** property:

```js
import React from 'react';
import { SpotifyEmbed } from 'spotify-embed'; // import the SpotifyEmbed component

function App() {
  return (
    <>
      <SpotifyEmbed src="https://open.spotify.com/playlist/37i9dQZF1DX5trt9i14X7j" />
      {/* pass an spotify resource URL/URI */}
    </>
  );
}

export default App;
```

It will parse the URL/URI and generate the embed:
